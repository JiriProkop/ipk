# Projekt 1
*Klient pro server kalkulačku.*

Funkcionalita a známé limitace obsaženy v souboru *CHANGELOG.md*
## Překlad a spuštění

### Překlad
``` make ```

Potřeba alespoň g++ verzi 10.
### Spuštění
```
./ipkcpc -h <host> -p <port> -m <mode> 
        <host>      IPv4 adresa serveru
        <port>      'port' serveru
        <mode>      "udp" nebo "tcp"

Př.:    ipkcpc -h 1.2.3.4 -p 2023 -m udp
```
